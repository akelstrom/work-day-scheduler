# Work Day Scheduler 

## Desciption
Work day scheduler made for user to schedule tasks in a business day and sets them into local storage when you press save.

## Tech Used
* HTML
* CSS
* JavaScript
* JQuery
* Bootstrap
* Moment.JS
* Font Awesome
* Google Fonts
